Welcome! Please use the following commands to navigate your file system.

LGIN - Login (ex. LGINusernamepassword)
LGOT - Logout (ex. LGOT)

STOR - Store file       (ex. STORfile.txt)
RETR - Retrieve a file  (ex. RETRfile.txt)
DELE - Delete a file    (ex. DELEfile.txt)
LIST - List all files   (ex. LIST)
